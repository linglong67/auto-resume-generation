# 이력서 자동생성 프로그램
#### 1. 개요
이력 정보와 자기소개서를 입력 받아서 Excel 파일을 생성하는 프로그램

#### 2. 구현 기능 목록
* 입력 받은 정보 저장
  - 개인 정보 저장
  - 학력 정보 저장
  - 경력 정보 저장
  - 자기소개서 저장
* 이력서 생성
  - 이력서 시트 생성
  - 자기소개서 시트 생성
  - Excel 파일 생성 
   
#### 3. 입력 & 출력 내용
* 입력 예시
  > 사진 파일명을 입력하세요:<br>
    이름을 입력하세요:<br>
    이메일을 입력하세요:<br>
    주소를 입력하세요:<br>
    전화번호를 입력하세요:<br>
    생년월일을 입력하세요 (예: 1990-01-01):
* 출력 예시
  > Excel 파일 참고

#### 4. 개발도구 & 기술스택
* IntelliJ IDEA
* Java 11
* Maven 4.0.0
* apache.poi 5.2.0
